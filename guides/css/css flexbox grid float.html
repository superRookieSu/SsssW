Flexbox

1-demensional layout system

Properties for flex container:
  • display: flex | inline-flex
  • flex-direction: row | row-reverse | column | column-reverse
  • flex-wrap: nowrap | wrap | wrap-reverse
  • flex-flow: flex-direction flex-wrap the shorthand.
  • justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly align all the flex items along the main axis.
  • align-items: flex-start | flex-end | center | stretch | baseline align all the flex item along the cross axis.
  • align-content: flex-start | flex-end | center | stretch | space-between | space-around align the flex container's lines the cross axis. For multi-lines
Properties for flex items:
  • order: number controls the order of flex item appear in the flex contianer.
  • flex-grow: number grow factor. default 0.
  • flex-shrink: number shrink factor when overflow. default 1.
  • flex-basis: length init size.
  • flex: none | flex-grow flex-shrink flex-basis the shorthand.
  • align-self: flex-start | flex-end broken align-items

Note: float clear vertical-align have no effect on a flex item.

Grid

2-demensional layout system

Properties for the grid container:
  • display: grid | inline-grid
  • grid-template-columns & grid-template-rows:
    • track size: can be a length(px), a per`centage(%), or a faction of the free space in the grid (using the fr unit)
    • line-name: arbitrary name. for example:
      grid-template-columns: [first] 40px [line2] auto [line2end]
      grid-template-columns: [first] 40px [line2end line3start] 30% [line-end] a line can have more than one name.
    • repeat: grid-template-column: repeat(3, 20px [line-start]);
      equivalent to: grid-template-column: 20px [line-start] 20px [line-start] 20px [line-start]
  • grid-template: grid-template-rows / grid-template-columns
    for example: grid-template: auto 50px auto / 25px auto 25px
  • grid-template-area: guides
  • grid-column-gap
  • grid-row-gap
  • grid-gap: grid-row-gap grid-row-gap the shorthand
  • justify-items
  • align-items
  • place-items
  • justify-content
  • align-content
  • place-content
  • grid-auto-columns
  • grid-auto-rows
