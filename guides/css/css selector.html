Attribute Selector

Introduction

Presence and value attribute selectors:
  • [attr]{}: have the attribute attr, whatever it's value;
  • [attr=val]{}: only attr === val;
  • [attr~=val]{}: only the attribute val is one of the 'space-separated' list;
Substring value attribute selectors:
  • [attr^=val]{}: attr must start with val;
  • [attr$=val]{}: attr must end with val;
  • [attr*=val]{}: value contains val;

combinators and selector lists

Introduction

combinators and selector lists:
  • A, B: A or B;
  • A B: B that is a descendant of A (B is child of A or A's child's child's child);
  • A > B: element mathing B that is a direct child of an element matching A;
  • A + B: B that is the next sibling of A;
  • A ~ B: B that is one of the next sibling A;

Example : A, B

#d1, .c1, div[name="n1"] {

Pseudo Selector
  • :first-child select the first element within another element
  • :last-child select the last element within another element
  • :nth-child(index) select the index element within another element(in order)
  • :nth-last-child(index) select the index element within another element from the back (reverse order)
  • :only-child select element that it is the only element inside of its parent
  • :only-of-type select the element of that type (the element is the only element inside its parent)
  • :first-of-type select the first element of that type within other element
  • :last-of-type select the last element of that type within other element
  • :nth-of-type()
    • :nth-of-type(index) select the index element of that type within other element
    • :nth-of-type(odd) select the odd element of that type within other element(1th, 3th, 5th...)
    • :nth-of-type(even) select the even element of that type within other element(2th, 4th, 6th....)
    • :nth-of-type(An+b) nth = b + A * n, n start at 0; (:nth-of-type(2n+1) ==> (1, 3, 5,...))
  • :empty select all empty elemenst of that type
  • :not(selector) select all elements that do not match selector. (:not(#someclass))
