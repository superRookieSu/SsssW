overflow

auto: content is hidden, scroll bars are shown
hidden: content is hidden, scroll bars are hidden

background-clip

background-clip: border-box;

border-box: the bg extends to the outside edge of the border
padding-box: the bg extends to the outside edge of the padding
content-box: the bg within the content box

margin collapsing

if two adjacent elements both have the margin set on them. and the two margins touch, the larger of the two remains, and the smaller one disappears. (effect on vertical, not horizontal)

background

background-color : rgba(255,255,255, 0.3); -default color is not white but transparent

background-image : - bg image

    ulr(path) : image path
    linear-gradient() : color gradients

background-repeat: repeat(default), for background-image

  • repeat : repeat both vertical and horizontal.
  • repeat-x : repeat horizontal
  • repeat-y : repeat vertical
  • no-repeat : will not repeat at all.

background-position : x y, for background-image

  • background-position: 20px, 20px
  • background-position: 20rem 20rem
  • background-position: 90% 20%
  • background-position: right center;

border

border: border-width border-style border-color;
border-radius: 20px; used like margin
  • border-radius: 20px; // all the corners use the same radius.
  • border-radius: 20px 10px; // 20px is top left and bottom right, 10px is top right and bottom left
  • border-radius: 20px 10px 30px 0; //you know that.
  • border-radius: 30px/20px 50px/ 30px; //ellilptical corner, use x-radius / y-radius;
border: 2px solid blue;
border-radius: 40px/20px;
border-image-source: url(xxx) ... border-image-slice: 40 ... border-image-repeat: stretch ...
  • stretch: the default, images are stretched to fill the border, not recommended.
  • repeat: the default, images are repeated util it to fill the border
    • ...
shorthand:
border-image: url, slice, repeat

box-shadow

box-shadow: 5px 5px 5px rgba(255,225,0,0.5);

four params in box-shadow
  1. horizontal offset
  2. vertical offset
  3. blur radius
  4. base color

inset: make the shadow become an inner shadow

outer shadow
box-shadow: 5px 5px 5px rgba(0,0,0,1)
inner shadow
box-shadow: inset 5px 5px 5px rgba(0,0,0,1)
